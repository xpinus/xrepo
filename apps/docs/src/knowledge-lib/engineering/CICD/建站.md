1. 购买服务器

在服务器上部署项目，先使用ip直接访问没问题

![](./asset/structure.jpg)
```docker-compose.yml
services:
    backend:
        container_name: backend_server
        build: ./backend
        image: backend:$BACKEND_COMMIT
        ports:
            - 7001:7001
        restart: always
    frontend:
        container_name: frontend_server
        build: ./frontend
        image: frontend:$FRONTEND_COMMIT
        ports:
            - 80:80
        restart: always
        depends_on:
            - backend
```
