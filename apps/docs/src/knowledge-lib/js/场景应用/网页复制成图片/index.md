# 网页复制成图片

将网页上的dom元素导出为图片，直接下载或者复制到剪贴板

## 原理

- 获取该区域的 DOM 元素
- 将 DOM 转为 Canvas
- 将 Canvas 转为 Blob
- 将 Blob 写入到剪贴板

> 你提到实现页面元素导出为图片，实现思维导图的到处下载，优化了第三方库
