# 【菟杞精芪平衡茶】
**——滋阴扶阳，脾肾同补，气血同调**

### 一、配方设计思路
本方以“阴中求阳，阳中求阴”为核心理念，不偏不火，旨在通过平补的方式，缓缓滋养先天之本（肾）与后天之本（脾），调和阴阳，适用于现代人因劳累、压力导致的亚健康状态，表现为精神不振、易疲劳、抵抗力弱、腰膝乏力，但无明显大寒大热之证者。

### 二、具体配方与用量（一日量）
| 药材 | 用量 | 性味归经 | 在方中的作用 |
| :--- | :--- | :--- | :--- |
| **黄精** | 10克 | 甘，平。归脾、肺、肾经。 | **君药**：平补气阴之核心，润而不腻，补脾益肾，填补精髓。 |
| **枸杞子** | 8克 | 甘，平。归肝、肾经。 | **臣药**：滋补肝肾，益精明目，助黄精养阴之效。 |
| **黄芪** | 6克 | 甘，微温。归脾、肺经。 | **臣药**：补气升阳，固表御邪，与黄精合用，气阴双补，提振功能。 |
| **菟丝子** | 6克 | 辛、甘，平。归肝、肾、脾经。 | **佐使药**：平补阴阳，补肾固精，养肝明目，其性平和，助阳而不燥。 |
| **陈皮** | 3克 | 辛、苦，温。归脾、肺经。 | **佐药**：理气健脾，燥湿化痰，防止诸滋补药过于壅滞，使补而不腻。 |

### 三、制法与服法
1.  **冲泡法（日常简便）**：
    *   将所有药材放入养生壶或保温性能好的杯中。
    *   注入沸水500-600毫升，加盖焖泡20-30分钟。
    *   可反复冲泡2-3次，代茶饮用，最后可将黄精、枸杞子嚼服。

2.  **煎煮法（功效更佳）**：
    *   将药材（陈皮可后下）放入砂锅，加冷水约800毫升，浸泡20分钟。
    *   大火煮沸后，转小火慢煎20-25分钟，滤出茶汤。
    *   可再次加水煎煮一次，两次茶汤混合，分次温热饮用。

**建议频次**：每日1剂，每周饮用3-5天为宜。可连续饮用1个月，观察身体反应。建议在白天饮用，尤以上午为宜，避免晚间服用后精力过旺影响睡眠。

### 四、配方特点
1.  **性味平和**：全方以“甘、平”为主，佐以微温、微辛，无大寒大热之品，犹如文火慢炖，补益而不峻烈。
2.  **阴阳双补**：
    *   **滋阴**：黄精、枸杞子滋养肾阴、肝血。
    *   **补阳**：菟丝子、黄芪在滋阴基础上微微生阳，取“阴为阳之基，阳为阴之使”之意。
3.  **脾肾同调**：黄芪、黄精、陈皮共健脾胃以化生气血；黄精、枸杞、菟丝子同补肾之阴阳。脾胃健运则补药能被有效吸收。
4.  **补中有行**：加入少量陈皮，理气运脾，使全方补而不滞，滋而不腻，易于人体接纳。

### 五、适宜人群
*   长期处于亚健康状态，易疲劳、精力不济者。
*   **用眼过度**的脑力劳动者、办公室人群。
*   中老年人日常**脾肾虚弱**的平补调理。
*   病后、产后体虚，处于缓慢恢复期者。
*   冬季畏寒、夏季乏力的**轻度阴阳两虚**人群。

### 六、注意事项与禁忌
*   **不适宜人群**：
    *   感冒发烧、身体有急性炎症（如咽喉肿痛、肠胃炎）者暂停。
    *   脾胃极度虚弱、经常腹胀、舌苔厚腻**湿浊重**者，应先健脾化湿。
    *   阴虚火旺（如五心烦热、盗汗严重）或实热证（如口苦、便秘、目赤）明显者慎用或调整配方。
    *   孕妇、儿童及有特定疾病者请在医师指导下使用。
*   **饮食配合**：饮用期间，饮食宜清淡均衡，避免过度油腻、生冷，以保证脾胃最佳吸收状态。
*   **观察反馈**：如饮用后出现口干、上火或腹胀，可减少黄芪用量至3克，或增加麦冬5克以佐制。
